# 🌱 Micro Garden – Automated Smart Irrigation System

## Overview

Micro Garden is an automated indoor smart irrigation system designed to monitor soil moisture and water plants automatically. The system uses multiple soil moisture sensors and water pumps to create independent watering zones and maintain optimal plant conditions.

## Features

- Automatic soil moisture monitoring  
- Multi-zone irrigation system  
- Timed watering control  
- Arduino-based embedded system  
- Low-cost and scalable design  

## How It Works

The system continuously monitors soil moisture levels. When the soil becomes dry, the corresponding watering system is activated for a fixed duration. This approach helps prevent overwatering and improves water efficiency.

## Purpose

This project was developed to explore embedded systems, sensor integration, and automation for smart indoor gardening applications.
